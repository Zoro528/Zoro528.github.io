aistation

发表于 2019-11-15

AIStation 环境搭建说明

ssh连接

  1. 需要开放ip，让本机ip可以访问集群
  2. 可让开放的ip配置内网环境（196.168. . ),让其余机器也可访问aistation集群。
  3. ssh连接，默认端口22，连接方式为ssh name@10.200.128.100,不用修改，默认进入mu01cpu集群环境，可通过ssh gpu01–ssh gpu07 进入到不同的gpu环境中。

python 环境安装

  1. conda安装，可选择miniconda或者anaconda，自带的python版本为3.7.4，也可自行安装其他python版本。安装后可自行更换conda源（如中科大源和清华源等。),下载速度慢也可配置pip源。
  1. 其他pypi包安装，如matplotlib、numpy等
  1. torch安装，查看cuda（10.1）和cudnn（7.1.4）版本，先要通过ssh进入gpu环境中

安装与cuda版本对应torch。选择pip或者conda

可能遇到的问题：
pytorch 安装时由于下载速度过快，磁盘写入速度太慢，可用wget限制下载速度，利用whl安装包来安装。

  1. jupyter notebook 配置，使得本低能够可视化访问服务器资源。（在vscode配置过程中，因为flock在nfs服务器上不能使用，故不选用vscode，可能以后会支持）

    一、pip 安装jupyter，pip install jupyter
    二、然后生成配置文件jupyter_notebook_config.py，jupyter notebook --generate-config
    三、打开Ipython终端，生成jupyter密码

    四、vim修改jupyter_notebook_config.py，vim ~/.jupyter/jupyter_notebook_config.py

    五、Xshell配置IP映射
    在右键属性中找到隧道，添加本地IP和端口与服务器端口的映射，然后本地可以使用

  1. 其他

    jupyter notebook 主题配置
    一. 安装Jupyter主题：pip install jupyterthemes
    二. 更改主题配色：jt -t monokai -f fira -fs 13 -cellw 90% -ofs 11 -dfs 11 -T -N,更多配置见jupyter-themes

    jupyter notebook 代码补全
    安装nbextensions插件：pip install jupyter_contrib_nbextensions，然后激活该插件jupyter nbextensions_configurator enable –user

    注：如果页面无Hinterland项，或者不全，命令行执行：
    jupyter contrib nbextension install --user --skip-running-check

文件传输

  • 常用命令有ftp、sftp、scp等

  • 安装Xftp，可传输文件夹或文件

  • 在jupyter notebook中可以传输单个文件或者压缩包。

  • 源码同步：git，pycharm
