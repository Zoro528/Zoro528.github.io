idea快捷键

Ctrl+D 复制当前行
Ctrl+Y 删除当前行
Shift+Enter 向下插入新行

vs code 快捷键

Shift + Ctrl + k 删除当前行
Ctrl+Enter 向下插入新行
Ctrl + b 显示/隐藏左侧目录栏
Ctrl + ~ 控制台终端显示与隐藏

退出python命令行

你可以执行CTRL-D (Windows 下是 CTRL-Z)来关闭编译器。

文件批量修改小技巧

通过os.listdir()来获取某一目录下所有的文件名(与之相关的还有os.walk()),然后通过字符串操作（如：切片）来获取想要的字段（正则也是一种方式），最后还可以根据os.rename()来修改文件名称。

docstring注释

通过使用"来注释函数，一般为三个"，且放在函数第一行，可以使用函数名.doc来输出，或者通过help(函数名)来输出。

有限状态机

有限状态机（finite-state machine,缩写FSM)是表示有限个状态以及在这些状态之间的转移和动作等行为的数学模型。FSM可以把模型的多状态、多状态间的转换条件解耦。可以使维护变得容易，代码也更加具有可读性，也更加艺术。
